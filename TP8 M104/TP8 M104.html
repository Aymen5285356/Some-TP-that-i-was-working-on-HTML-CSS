Énoncé 1: Création d'un formulaire de contact stylisé

Consignes :

  1. Créez un formulaire de contact HTML avec les champs suivants : Nom, Adresse e-mail, Message
  2. Utilisez CSS pour styliser le formulaire avec des styles pour les champs, étiquettes, bouton et animations
  3. Ajoutez des pseudo-classes CSS pour créer des effets de survol ou de focus
  4. Assurez-vous que le formulaire est bien aligné et correctement espacé

Énoncé 2: Création d'un menu de navigation déroulant

Consignes :

  1. Créez un menu de navigation horizontal avec Accueil, Produits, Services, Contact
  2. Utilisez des pseudo-classes CSS pour créer des effets de survol
  3. Créez un sous-menu déroulant pour "Produits" avec au moins trois liens
  4. Stylez le sous-menu avec des effets au survol
  5. Assurez-vous que le sous-menu disparaît quand on quitte la zone
